# 🚀 Run the Development Server

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

# 🛠️ Build the Project

```bash
npm run build
# or
yarn build
# or
pnpm build
# or
bun build
```

# 🌍 Deploy Locally

```bash
npm run start
# or
yarn start
# or
pnpm start
# or
bun start
```
